# ¿Qué necesitamos instalar?
Para programar en Python necesitas dos cosas:
- **Python** → el intérprete, el que "entiende" tu código.
- **VS Code** → el editor, donde vas a escribir tu código. Esto para programas un poco más avanzados, para aprender las bases usaremos el IDLE.
---
# Instalar Python
1. Ve a **python.org/downloads**
2. Descarga la versión más reciente
3. Ejecuta el instalador
4. ⚠️ **Marca la casilla "Add Python to PATH"** antes de continuar
5. Clic en "Install Now"

Para verificar que se instaló bien, abre la terminal y escribe:
```bash
python --version
```
Si ves algo como `Python 3.x.x` →  **LISTO**.
---
# Instalar VS Code
1. Ve a **code.visualstudio.com**
2. Descarga e instala normalmente
3. Abre VS Code

Para trabajar más comodamente con este editor de código activa las siguientes casillas en la instalación:

- [x] Agregar la acción "Abrir con Code" al menú contextual de archivos de Windows Explorer.

- [x] Agregar la acción "Abrir con Code" al menú contextual de directorios de Windows Explorer.

# Configurar VS Code para Python
Dentro de VS Code:
1. Ve a la sección de **Extensiones** (ícono de bloques en la barra lateral)
2. Busca **"Python"** (la extensión de Microsoft)
3. Instala

Eso es todo. VS Code ya sabrá que vas a trabajar con Python.
