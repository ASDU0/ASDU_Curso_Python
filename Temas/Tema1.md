# ¿Qué es programar?

Programar es el proceso de crear instrucciones que una computadora puede entender y ejecutar para realizar tareas específicas.

Estas instrucciones se escriben utilizando lenguajes de programación como Python, C++, Java o JavaScript.

El objetivo de la programación es resolver problemas mediante algoritmos y lógica computacional.

---

# ¿Cómo funciona la programación?

La programación funciona mediante una secuencia de instrucciones que la computadora ejecuta paso a paso.

El proceso general es:

1. El programador escribe código.
2. El compilador o intérprete traduce el código.
3. La computadora ejecuta las instrucciones.
4. Se obtiene un resultado.

# Lenguajes de programación

Existen muchos lenguajes de programación, cada uno diseñado para diferentes objetivos.

## Algunos ejemplos:

- Python → Inteligencia artificial, automatización, ciencia de datos.
- C++ → Videojuegos, sistemas operativos.
- JavaScript → Desarrollo web.
- Java → Aplicaciones empresariales y Android.
